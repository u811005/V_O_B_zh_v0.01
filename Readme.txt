1.1 仮タイトル
Village of Bacchus（仮）

1.2. バックストーリー

主人公は豊穣と混沌の神バッカス。
「新しき神」に敗れ、永い眠りについていた彼が目覚めたのは 小さな開拓村だった。
既に忘れ去られて久しく、神力も殆ど失われていた彼だったが豊穣心としての本能と神の力を取り戻すため、
この地にある魔素を利用して奇跡を行使し、村の発展を手助けすることを決意する。

1.3. 序盤の進め方
まず村人の「仕事」を設定します。
食料と資材を集めながら、村の発展を目指してください。
村人は基本的に「仕事」で設定した行動を行いますが、
プレイヤーが別の「行動」を指示することも可能です。
村人は体力やメンタルが減少すると自動で休養します。
勤勉が高い村人ほどギリギリまで頑張る傾向にあります。

名前をクリックすると会話画面になります。
村人の話を聞いたり、仕事の任命や訪問者の勧誘を行うことができます。

右下のボックスにて、仕事や用語の詳細を検索することが可能です。

農作業、醸造、漁の最中に、まれに奇妙な秘宝が見つかることがあります。
行商人が秘宝を持っている場合は、資金500で購入できます。

1.4. その他
ローカル保存とjsonでの保存ができますが、
バージョンアップに伴いセーブファイルが使えなくなる可能性があります。

1.5. 仕事の適性
各仕事には適正ステータスがあります。

仕事の消費は大きく三種類に分かれます。
体力系…体力24、メンタル12を基準に消費
バランス系…体力18、メンタル18を基準に消費
頭脳労働系…体力12、メンタル24を基準に消費

農作業…耐久＊勤勉（体力系）
狩猟…筋力＊勇気（体力系）
漁…耐久＊勇気（体力系）
伐採…筋力＊勤勉（体力系）
採集…器用＊知力（バランス系）
内職…器用＊勤勉（頭脳労働系）
行商…魅力＊知力（バランス系）
研究…魔力＊知力（頭脳労働系）
警備…筋力＊倫理（体力系）
看護…魔力＊倫理（バランス系）
詩人/踊り子…魅力*魅力/魅力＊好色（バランス系）
神官/シスター…魅力*倫理（頭脳労働系）
写本…器用＊知力（頭脳労働系）
錬金術…魔力＊知力（バランス系）
機織り…器用＊勤勉（バランス系）
醸造…魔力＊勤勉（バランス系）
巫女/バニー…魅力・魔力・好色/魅力＊好色（バランス系）

迎撃…物理攻撃（筋力*勇気ー相手の耐久）or 魔法攻撃（魔力＊勇気）
　　　ダメージが多い方を自動選択

罠作成…器用＊知識　最初のターンに攻撃して離脱

詳細は右下の「キーワードを入力」欄に検索ワードを入れて確認してください。
