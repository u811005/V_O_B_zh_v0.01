1.1 仮タイトル
Village of Bacchus（仮）

1.2. バックストーリー

主人公は豊穣と混沌の神バッカス。
「新しき神」に敗れ、永い眠りについていた彼が目覚めたのは 小さな開拓村だった。
既に忘れ去られて久しく、神力も殆ど失われていた彼だったが豊穣心としての本能と神の力を取り戻すため、
この地にある魔素を利用して奇跡を行使し、村の発展を手助けすることを決意する。

1.3. 序盤の進め方
まず村人の「行動」を設定します。
食料と資材を集めながら、村の発展を目指してください。
食料と資材には保管上限があり、初期上限は400です。納屋や貯蔵庫で上限を増やせます。
村人は基本的に「行動」で設定した内容を実行しますが、
プレイヤーが別の「行動」を指示することも可能です。
村人は体力やメンタルが減少すると自動で休養します。
勤勉が高い村人ほどギリギリまで頑張る傾向にあります。

名前をクリックすると会話画面になります。
村人の話を聞いたり、行動の任命や訪問者の勧誘を行うことができます。

右下のボックスにて、行動や用語の詳細を検索することが可能です。

農作業、醸造、漁の最中に、まれに奇妙な秘宝が見つかることがあります。
行商人が秘宝を持っている場合は、資金500で購入できます。
村史では、村の発展や村人の出入り、婚姻、出生、肉体交換などの主要な出来事を確認できます。
村人一覧などの記録ボタンから、各人物に関わる出来事を個人記録として確認できます。

1.4. その他
ローカル保存とjsonでの保存ができますが、
バージョンアップに伴いセーブファイルが使えなくなる可能性があります。

1.5. 行動の適性
各行動には適正ステータスがあります。

行動の消費は大きく三種類に分かれます。
体力系…体力24、メンタル12を基準に消費
バランス系…体力18、メンタル18を基準に消費
頭脳労働系…体力12、メンタル24を基準に消費

農作業…耐久＊勤勉（体力系）
狩猟…筋力＊勇気（体力系）
漁…耐久＊勇気（体力系）
伐採…筋力＊勤勉（体力系）
採集…器用＊知力（バランス系）
内職…器用＊勤勉（頭脳労働系）
行商…魅力＊知力（バランス系）
研究…魔力＊知力（頭脳労働系）
警備…筋力＊倫理（体力系）
看護…魔力＊倫理（バランス系）
詩人…魅力＊知力（バランス系）
踊り子…魅力＊好色（バランス系）
神官/シスター…魅力*倫理（頭脳労働系）
写本…器用＊知力（頭脳労働系）
錬金術…魔力＊知力（バランス系）
機織り…器用＊勤勉（バランス系）
醸造…魔力＊勤勉（バランス系）
巫女/バニー…魅力・魔力・好色/魅力＊好色（バランス系）

迎撃…前衛。物理攻撃（筋力*勇気ー相手の耐久）or 魔法攻撃（魔力＊勇気）
　　　ダメージが多い方を自動選択

籠城…前衛。防壁建設で解放。攻撃せず、被ダメージ0.8倍。攻撃された時は反撃

射撃…中衛。櫓建設で解放。器用＊勇気 - 相手の耐久*1.5。反撃を受けない

罠作成…後衛。器用＊知力　0ターン目に攻撃して離脱

襲撃予約…月初判定で当選しても当月即時発生せず、翌月の襲撃として内部予約されます。
　　　　　襲撃後の翌月は予約判定なし。2か月目以降は10%から毎月1.45倍、通常最大75%まで上昇します。
　　　　　荒廃中は襲撃予約確率が2倍になり、最大100%になります。
　　　　　太陽の巫女がいる場合、予約済み襲撃を60%で「太陽神の予言」として事前通知します。

詳細は右下の「キーワードを入力」欄に検索ワードを入れて確認してください。
