# PROJECT_OVERVIEW.md

`Village of Bacchus` は、神バッカスとして小さな開拓村を育てるブラウザ向け村運営シミュレーションです。プレイヤーは村人に仕事や行動を割り振り、資源を集め、建築や奇跡を使いながら村を存続・発展させます。

世界観・ストーリーの詳細は `WORLDVIEW.md` を参照してください。今後、外部勢力、宗教、奇跡、襲撃者、イベントなどを追加する場合は、同文書の設定を参考にします。

## プロジェクトの特徴

- 静的な HTML / CSS / JavaScript で動くゲーム。
- `index.html` から ES module の JavaScript を読み込む構成。
- サーバー側処理やビルド工程は持たない。
- セーブは JSON ファイル保存と localStorage 保存に対応している。
- 村人は肉体情報と精神情報を別々に持ち、奇跡や成長で状態が変化する。
- 会話は口調、状態、仕事、季節、イベントなどから候補を選ぶ仕組みを持つ。

## コアループ

1. 村人の仕事と行動を決める。
2. 必要に応じて建築、奇跡、勧誘、保存/読込を行う。
3. `次の月へ` で月次処理を進める。
4. 仕事効果、固定イベント、ランダムイベント、妊娠・出産・成長、襲撃判定などが処理される。
5. 村人がいなくなるとゲームオーバーになる。

## 主要資源

- `food`: 食料。村人の生活維持に関わる。
- `materials`: 資材。建築や冬の消費に関わる。
- `funds`: 資金。建築や取引などで使う。
- `mana`: 魔素。奇跡の行使に使う。
- `fame`: 名声。訪問者や村の発展に関わる。
- `tech`: 技術。生産や発展要素に関わる。
- `security`: 治安。襲撃や村の安全に関わる。

## 主なシステム

- 村人生成: 初期村人、訪問者、子供、イベント由来の人物を生成する。
- 仕事: 村人の能力値、仕事、趣味、状態から資源や状態変化を処理する。
- 行動: 通常行動、休養、襲撃中の迎撃・罠作成などを扱う。
- 建築: 資源を消費して村の機能や上限を拡張する。
- 奇跡: 神の力として、回復、交換、勧誘、成長、特殊効果などを扱う。
- 会話: 口調、状態、仕事、季節、訪問者、イベントに応じたセリフを表示する。
- 人間関係: 恋人、配偶者、親子などの関係を管理する。
- 繁殖・成長: 妊娠、出産、産褥、子供の成長、成人化を扱う。
- ランダムイベント: 村の状況に変化を与える月次イベントを処理する。
- 襲撃: 襲撃者、迎撃、罠作成、戦闘ログ、防衛結果を扱う。
- 秘宝: 農作業・醸造・漁や行商人取引から入手できる使い切りの特別な品を扱う。
- 保存/読込: JSON ファイルと localStorage の両方を扱う。

## 画面構成

- 村情報: PC表示では見出し部分に規模称号を表示し、スマホ仮表示では従来どおり村情報欄の先頭ボックスに規模称号を表示する。年月、資源、警告、主要操作を扱う。
- 規模称号: 規模に応じて、村の外部的な呼び名を表示する。
- 村人一覧: 村人の顔、能力、状態、仕事、行動、詳細。
- 訪問者一覧: 勧誘候補を表示する。
- 襲撃者一覧: 襲撃中の敵情報を表示する。
- ログ: 月次処理やイベント結果を表示する。
- 用語検索: 仕事、資源、状態などの説明を検索する。
- 各種モーダル: 会話、奇跡、建築、秘宝、交換、襲撃、ランダムイベントなど。

## 実行方法

このプロジェクトは静的ファイルで構成されています。ブラウザが `file://` での ES module 読み込みを許可する場合は `index.html` を直接開けます。拒否される場合は、リポジトリのルートを静的サーバーとして配信して `index.html` を開いてください。

依存関係のインストールやビルドは不要です。
